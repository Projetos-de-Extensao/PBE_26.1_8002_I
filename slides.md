#  Apresentação de Slides para a AP1 

##  Descrição
O presente arquivo tem como objetivo principal formalizar a apresentação de slides acerca do material referente à AP1. Esta apresentação funciona como um resumo detalhado e estratégico das etapas de iniciação e elaboração, que são fundamentais para o projeto. Essas fases iniciais foram essenciais para criar uma base sólida e segura, garantindo que o desenvolvimento do nosso sistema siga um caminho bem planejado e esteja pronto para os próximos desafios.
